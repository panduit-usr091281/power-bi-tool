# Infographic Visual Element Reference Guide

A comprehensive list of graphical representations used in modern infographics. Use this reference to select and combine visual techniques when constructing infographic-style data visualizations.

---

## 1. Proportional & Quantitative Elements

| Element | Description | Best For |
|---------|-------------|----------|
| **Pictogram Arrays** | Grids of repeated icons where filled count represents quantity | Showing counts, percentages, survey results |
| **Proportional Circles** | Circles sized by area to represent relative values | Comparing magnitudes across categories |
| **Proportional Squares** | Squares or rectangles scaled to value | Budget breakdowns, area comparisons |
| **Waffle Charts** | 10×10 grids with colored cells representing percentages | Showing parts of 100% |
| **Bubble Clusters** | Grouped circles of varying sizes packed together | Multi-variable comparisons |
| **Stacked Coins/Discs** | Stacked circular elements where height = value | Financial data, accumulated totals |
| **Thermometer/Level Fill** | Vertical or horizontal containers filled to a level | Progress toward goals, capacity |
| **Battery Indicators** | Battery-shaped containers with partial fill | Charge levels, completion status |
| **Progress Rings** | Circular arcs (donut-style) showing completion | KPIs, percentage completion |
| **Gauge Dials** | Semicircle meter with needle pointing to value | Performance scores, ratings |

## Usage Guidelines

1. **Combine 2-3 element types** per infographic for visual variety without chaos
2. **Maintain consistent style** (don't mix flat icons with realistic illustrations)
3. **Every visual element must encode data** — avoid purely decorative elements that don't inform
4. **Size and color are your primary data encoders** — make differences obvious
5. **Always include readable numeric labels** near visual elements
6. **White space is a design element** — don't overcrowd
7. **Visual hierarchy guides the eye:** largest/boldest first, details second
8. **Limit to 5-7 colors maximum** per infographic
9. **Test readability:** if you remove the illustrations, can you still read the data?
10. **Accessibility:** ensure sufficient contrast ratios and don't rely solely on color
